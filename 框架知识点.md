#### vue 列表的 key

key 在列表渲染中的作用是：在复杂的列表渲染中快速准确的找到与 newVnode 相对应的 oldVnode，提升 diff 效率。
vue 中使用 key 不一定能提提高 diff 效率。例如有个列表

```html
<!-- 初始 -->
<li>A</li>
<li>B</li>
<li>C</li>

<!-- 打乱顺序 -->
<li>C</li>
<li>A</li>
<li>C</li>
```

没有设置 key 时，vue 只需要更新 dom 的文本内容，如果给每个 li 设置了 key，vue 需要去对比新旧然后进行 dom 的移动、添加或删除，效率反而更低。
